Module Organization

Piccolo.jl is organized into three main modules:

ModuleContents
QuantumSystems, trajectories, pulses, operators, isomorphisms
ControlProblems, objectives, constraints, integrators, templates
VisualizationsPlotting and animation utilities

See the Library Reference page for full docstrings of all exported types and functions.

Quick Reference

Creating Systems

# Basic system
sys = QuantumSystem(H_drift, H_drives, drive_bounds)

# From templates
sys = TransmonSystem(levels=3, δ=0.2, drive_bounds=[0.2, 0.2])

Creating Trajectories

# Unitary gate synthesis
qtraj = UnitaryTrajectory(sys, pulse, U_goal)

# State preparation
qtraj = KetTrajectory(sys, pulse, ψ_init, ψ_goal)

Creating Problems

# Smooth pulse optimization
qcp = SmoothPulseProblem(qtraj, N; Q=100.0)

# Time-optimal control
qcp_mintime = MinimumTimeProblem(qcp; final_fidelity=0.99)

# Robust optimization
qcp_robust = SamplingProblem(qcp, systems)

Solving and Analysis

# Solve
solve!(qcp; max_iter=100)

# Analyze
fid = fidelity(qcp)
traj = get_trajectory(qcp)
